
# Diabetes Risk Classification

This project uses machine learning algorithms to predict whether a patient is likely to have diabetes based on medical and health-related features.

## Dataset Features
- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age

## Objective
To compare multiple classification models and identify the best algorithm for diabetes prediction.

## Models Used
- Naive Bayes
- KNN
- Decision Tree
- Random Forest
- SVM
- Logistic Regression
- ANN
- XGBoost

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix

## Best Result
Random Forest achieved the highest performance with 79% accuracy.
